# Android Calculator App

A simple Android calculator built in Java using Android Studio.

## Features
- Basic arithmetic (+, −, ×, ÷)
- Clear button
- Equals calculation
- Advanced mode with calculation history

## Structure
- `MainActivity.java` – Handles UI interactions and button logic
- `Calculator.java` – Core calculation and history logic
- `activity_main.xml` – UI layout
